# BSOSkaner
Panel sterowania skanera podatności OpenVAS wykorzystujący własną bibliotekę komunikującą się ze skanerem protokołem GMP. 

## Funkcje
- Tworzenia oraz wysoka customizacji skanów, celów, skanowanych portów
- Zarządzania zadaniami (pauza, wznowienie, zatrzymanie)
- Dodawanie inetrwałów skanów
- Automatyczne raportowanie email po zakończonym skanie
- Graf przedstawiający siatkę hostów w sieci
